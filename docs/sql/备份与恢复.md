## 创建用户

```
CREATE USER 'backup_user'@'%' IDENTIFIED BY 'Goze@2026';
GRANT ALL PRIVILEGES ON *.* TO 'backup_user'@'%';
flush privileges ;
```

> --threads 4 可按照cpu数进行修改

## 备份

```
mydumper -u backup_user -p Goze@2026 -h 10.20.17.34 --regex '^asr.*' --verbose=3 -o backup 
```

## 恢复

```
myloader -u backup_user -p Goze@2026 -h 172.19.102.73 --verbose=3 -d backup 
```
