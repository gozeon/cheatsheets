

## md5

see [crypto-js](https://www.npmjs.com/package/crypto-js)

```js
const Crypto = require('crypto-js')

function md5(str) {
    return Crypto.MD5(str).toString()
}
```


## rsa

see [JSEncrypt](https://www.npmjs.com/package/jsencrypt)

```js
const jc = new JSEncrypt()
jc.setPublicKey("")
jc.setPrivateKey("")

// 加密
jc.encrypt("xxx")

// 解密
jc.decrypt("xxx")
```
